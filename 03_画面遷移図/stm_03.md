## インテリアECサイト
```uml
@startuml
[*]->トップページ
トップページ->商品検索結果: 検索
トップページ-right->商品一覧: 商品一覧をクリック
商品一覧->商品詳細: 商品をクリック
商品検索結果-down->商品詳細: 商品をクリック
商品詳細-->カート内: カートへをクリック
トップページ-left->お気に入り: お気に入りをクリック
お気に入り->商品詳細: 商品をクリック
トップページ-left->適性診断: 適性診断をクリック
適性診断->商品詳細: 商品をクリック
トップページ-->カート内: カートをクリック
カート内->トップページ: トップページへ戻るをクリック
カート内->商品詳細: 商品をクリック

state カート内{
[*]->カート
カート-->注文: 注文に進むをクリック
  state 注文{
    [*]->お届け先入力: 注文に進むをクリック
    注文->カート: カートをクリック
    お届け先入力->支払方法選択: 次へをクリック
    支払方法選択->お届け先入力: 戻るをクリック
    支払方法選択->最終確認: 注文内容を確認にクリック
    最終確認->支払方法選択: 戻るをクリック
    最終確認->購入完了: 注文確定をクリック
  }
}

state ヘッダー{
}
note left of ヘッダー: 全画面共通の遷移


ヘッダー-down->トップページ: ロゴをクリック
ヘッダー-down->ログイン: ログインをクリック
ヘッダー-down->会員登録: 会員登録をクリック
ヘッダー-down->適性診断: 適性診断をクリック
ヘッダー-down->お気に入り: お気に入りをクリック
ヘッダー-down->カート内: カートをクリック
ヘッダー-down->ヘルプお問い合わせ: ヘルプお問い合わせをクリック
@enduml
```
