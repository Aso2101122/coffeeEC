## Beginers coffee
```uml
@startuml
[*]->トップページ
トップページ->商品検索結果: 検索
トップページ-right->商品一覧: 商品一覧をクリック
商品一覧->商品詳細: 商品をクリック
商品検索結果-down->商品詳細: 商品をクリック
商品詳細-->カート: カートへをクリック
トップページ--up>お気に入り: お気に入りをクリック
トップページ-->カート: カートをクリック
カート->商品詳細: 商品をクリック

state ヘッダー{
}
note left of ヘッダー: 全画面共通の遷移


ヘッダー-down->トップページ: ロゴをクリック
ヘッダー-down->商品一覧: 豆/器具をクリック
ヘッダー-down->会員情報系: ログイン/新規会員登録/会員情報変更
ヘッダー-down->お気に入り: お気に入りをクリック
ヘッダー-down->カート: カートをクリック

state 会員情報系{
新規会員登録 : do/会員登録
ログイン : do/ユーザー認証
会員情報変更 : do/会員情報変更
}
@enduml
```
